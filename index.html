Installation et configuration de Media Station X

1. Allez dans le "magasin" de votre appareil et téléchargez l'application "Media Station X"
2. Lancez l'application Media Station X
3. Ouvrez les paramètres (Settings) - (Start Parameter) - (Setup)
4. Entrez l'adresse du playlist dans le champ : nnmtv.fun ou msx.nnmtv.fun ou nnmtv.github.io
5. C'est tout !
